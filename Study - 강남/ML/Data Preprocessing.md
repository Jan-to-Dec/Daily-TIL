## 1. 결측치 처리
결측치 (NaN) - 누락된 값, 비어 있는 값, Null

데이터 수집 과정에서 발생한 오류 등으로 결측치가 포함되는 경우 이 결측치를 정제하는 과정이 지나야 분석 결과에 왜곡이 없다. 

## 결측치 발견 & 제거
전체 칼럼 단위 결측치 확인
1. df.isnull().any()
2. df.isnull().sum() 

결측치제거
1. new_Df = df.dropna(subset=[' column name'])
2. new_df.shape

결측치(Na) 대체
1. 상수 대체 : 0 또는 특정 상수
2. 중위수 대체
3. 평균 대체
4. 최빈수 대체
