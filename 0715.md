## anaconda 명령어 정리
가상환경 목록
conda env list

가상환경 생성
conda create -n <가상환경명>
conda create -n <가상환경명> python=<버전> 

활성화 및 비활성화
conda activate <가상환경명>
conda deactivate

가상환경 삭제
conda env remove -n <가상환경명>

현재 가상환경에 패키지 목록 확인
conda list

현재 가상환경에 패키지 설치하기 
conda install <패키지명>

현재 가상환경에 패키지 제거하기
conda uninstall <패키지명>
